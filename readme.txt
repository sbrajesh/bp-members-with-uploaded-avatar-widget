Provides a widget to list members who uploaded a new profile photo(avatar)
== Description ==
The plugin provides a widget which lists the members who uploaded avatars on a BuddyPress based social network.
The members avatars list can be sorted on the basis of popularity/joining time/alphabetically/random.

For more details, please have a look here 

== Installation ==

1. Upload `bp-members-with-uploaded-avatars.zip` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Visit Appearance->Widgets and add the Members with Uploaded Avatars Widget to your sidebar
